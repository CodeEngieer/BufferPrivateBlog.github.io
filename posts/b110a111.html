HTML知识体系：

基本形式：

<!DOCTYPE html>
<html>
    <head>
        <title>这是元素标题</title>
    </head>
<body>
    <!--网页主题内容-->
    </body>

Font：字体

常用API：

  1. color：文字显示颜色
  2. size：文字显示大小

基于物理样式的文本元素：

  1. 加粗文字：< b> bold 加粗
  2. 放大元素：< big>
  3. 缩小元素：< small>
  4. 斜体元素：< i> italics
  5. 下标元素：< sub>
  6. 上标元素：< sup>
  7. 等框元素：< tt>
  8. 删除字：< del>

基于内容样式的文本元素：

  1. 强调元素：< em>
  2. 文字加粗：< strong>
  3. 提取元素：< samp>
  4. 首字母缩写符：< acronym>
  5. 变量显示元素：< var>
  6. 文献参考元素：< cite>

Image：图片

  1. border：边框
  2. align：对齐方式
  3. src ：文件地址
  4. alt：图片未加载替换文本
  5. style：浮动图像位置（可用css替换） style = “float:left” / style = “float:right”
  6. usemap：定义图像名称

        </body>
    </head>    
</html>

href：链接

< a> 标签定义超链接，用于从一个页面链接到另一个页面。

< a> 元素最重要的属性是 href 属性，它指定链接的目标。

在所有浏览器中，链接的默认外观如下：

  • 未被访问的链接带有下划线而且是蓝色的
  • 已被访问的链接带有下划线而且是紫色的
  • 活动链接带有下划线而且是红色的

父类属性：

  1. href 规定链接目标url
  2. download：指定下载链接
<a href = "图像url" download = "图像url">
<a href = "网址url">

  <source src = "movie.ogg" type = "video/ogg">

音频元素：< audio>

  1. autoplay：音乐就绪后马上播放
  2. loop：循环播放
  3. controls：显示音频控件
  4. src 文件url
 <audio controls>
 <source src = "horse.ogg" type = "audio/ogg">
 <source src = “horse.mp3" type = "audio.mpeg">
</audio>

无序列表：< ul>

列表项：< li>

常用API：

  1. compact：规定列表呈现效果比正常情况更小巧
  2. type：规定列表的项目符号的类型

    • circle：空心圆
    • square：实心方块

marquee：移动标签

常用API：

常用API 释义
direction 表示滚动的方向（默认为left）
behavior 表示滚动的方式
loop 决定滚动文字的滚动次数
scrollamount 决定文字滚动的速度
scrolldelay 决定文字滚动的延时
bgcolor 文字滚动范围的背景颜色
align
valign		 水平对齐/垂直对齐
width
height		 决定滚动文字在页面中的矩形范围大小
onmouseover = this.stop();
onmousemout =this.start();		 表示当鼠标移上区域的时候滚动停止

代码示例：

<!DOCTYPE html>
<html lang="en">

	<title>Document</title>
</head>
<body>
	<marquee direction="up" scrollamount="1" onmouseover="this.stop()" onmouseout="this.start()"><font size="2" color="#ff0000">四六级考试考试报名</font></marquee><

Form：表单

父类属性：

属性值 释义
type 规定显示< input> 元素控件
name 用在JavaScript中引用元素，或者在表单提交后引用表单数据，注意：只有设置name属性表单元素才能提交表单时传递他们的值
action 数据发送给指定服务器 < form action = “服务器名称”>，常用的服务器有PHP、JSP、ASP.Net
value 初始显示文字
width、height 控件长宽
size 长度

Table：表格（完成）

表格：< table>

  1. bordercolor：颜色边框
  2. border：边框属性
  3. cols：列属性
  4. cellspacing：单元格空白
  5. cellpadding：单元格间距
  6. align：对齐方式
  7. bgcolor 背景颜色

表格常用元素

  1. < th>：表格表头 tabel head
  2. < tr>：表格行 table row
  3. < td> 单元格 table data
    • colspan：单元格可横跨行数
    • rowspan：单元格可纵跨行数
    • align：对齐方式
    • width/ height：宽度
    • cellspacing：单元格空白
    • cellpadding：单元格间距

Frames：内联框架

框架：< frame>

常用API：

  1. frameborder：框架周围是否显示边框
    • 0/1：frameborder =”1”
  2. marginheight：框架上方和下方的比啊努努
  3. marginwidth：框架左侧和右侧边距
  4. name：规定框架名称
  5. noresize：规定无法调整框架大小
  6. scrolling：框架中是否显示滚动滚动条
    • yes/no/auto：
  7. src：规定框架中显示的文档的url

代码示例：

<!DOCTYPE html>
<html>
    <head>
        <title>frame框架功能测试</title>
    </head>
<body>
    <frameset rows="20%,*" scrolling ="yes">
    	<frame src="top.html" />
        <!--  <frame src="01.html" /> -->
    	 <frameset cols="20%,*">     <!--此部分内容替换了上面frame的位置内容-->
        	<frame src="left.html" />
            <frame src="right.html" />
        </frameset> 
    </frameset>
    </body>
</html>

不支持框架显示文本< noframe>
