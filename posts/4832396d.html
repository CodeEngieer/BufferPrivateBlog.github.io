CSS知识体系：

阅读本文前先看：

  1. 基本格式
<!DCOTYPE html>
<html>
    <head><!--头文件用于引入css文件-->
        <title>Demo功能代码</title>
     <style>  <!--引入修饰css部分-->
         p{
             text-align:center;
         }   
        </style>
    </head>
    <body> <!--代码主体内容-->
        <p>这是一段受css影响的文字</p>
    </body>
</html>
  1. CSS介绍：

修饰类型语言分支之一，相似语言有Unity中Shader着色器、Animation动画等，主要修饰html内的元素。属于前端开发语言。是用来表现HTML或XML等文件样式计算机语言，既可以修饰静态网页，有可以配合各种脚本语言动态的对网页各元素进行格式化。

  1. 元素修饰作用已在开头给出，*修饰为常用元素，请读者多注意！

选择集：

作用：相当于后端语言中JAVA中import、C# 中using导包方式

import java.util.*;  //JAVA引入外部文件
using System.Collections.Generic;  //C#引入外部文件
//Css引入
方式一：
<style>@import url(Style.css);</style>

方式二：（最常使用）
<link rel="Stylesheet" type ="text/css" href="Style.css"></link>

方式三：内联方式
<div style ="background:red"></div>

方式四：内嵌方式只能改变当前标签的样式，可维护性查
<!DOCTYPE html>
<html>
<!--头部部分相当于后端语言中导包部分-->
<head>
<style> 
 .conment{
    background-image:url("");
    background-repeat:none;
 }
  </style>
</head>
</html>

选择器：只想您需要设置样式的HTML元素

声明块：包含一条或多条用分号分隔的什么

  1. 每条声明都包含一个Css属性名称和一个值，以冒号分隔。
  2. 多条Css声明用分号分隔，声明块用花括号括起来。

选择器类型：

  1. id选择器：可以标记为特殊id的HTML元素指定样式，要选择具有特定id元素后跟元素，不能以数字开头
#{
   
}
  1. 类选择器：选择特定class属性HTML，选择特定class元素，句点字符，后面跟类名。
.center{
   text-align:center;
   color:black;
}
<h1 class = "center">受影响的红色标题</h1>
<p class = "center'> 受影响的红色段落</p>
  1. 通用选择器：*选择页面上所有HTML元素，页面内所有元素均受影响：
*{
  text-align:center;
  color:blue;
}
  1. 分组选择器：
h1,h2,p{
    text-align:center;
    color:red;
}

Style（样式）：

background（背景）：（完成）

background-image：背景图片

常用方法：

  1. url(URL)：图像的URL地址
  2. none：无图像背景会显示，这是默认
  3. inherit：指定背景图像从父元素继承

常用方法：

  1. color：选择指定颜色

  3. inherit：指定背景颜色，应该从父元素继承

background-position：背景图像位置

  1. 定位方式：
  • 页面背景图像：background-positon:center;
  • %定位背景图像：background-position:50%,20%;
  • 像素定位背景图像：background-position:50px,20px;
  1. inherit：继承父元素

background-repeat：背景图像重复

常用方法：

  1. repeat：背景图像将向垂直和水平方向重复
  2. repeat-x：只有水平位置会重复背景图像
  3. repeat-y：只有垂直位置会重复背景图像
  4. no-repeat：图像不会重复
  5. inherit：从父元素获取

background-attachment：背景图片固定

常用方法：

  1. scroll：背景图片随页面其余部分滚动（默认）
  2. fixed：背景图像是固定的
  3. local：背景图片随滚动元素滚动

background-clip：背景绘制区域：

  1. border-box：背景绘制在内容方框内，背景绘制在内容方框内（剪切成内容方框）。
  2. padding-box：背景绘制在衬距方框内（剪切成衬距方框）。
  3. border-box：默认值。背景绘制在边框方框内（剪切成边框方框）。

border（边框）：

border-image：边框样式图像填充

border-image-width：图像宽度

  1. number：表示相应border-width倍数
  2. %：边界图像区域的大小
  3. auto：自适应
border-image-source：图像引用
  1. image：边框使用图像路径
  2. none：默认，没有图像使用
border-image-repeat：是否重复：
  1. stretch：默认值，拉伸图像来填充区域
  2. repeat：平铺来填充区域
  3. round：对图像进行缩放以适应区域
  4. space：扩展控件分布图像周围

border-right/left：左/右边框属性

border-right/left-style：左右边框样式
  1. none：无边框
  2. dotted：定义点状边框（浏览器中为实线）
  3. dashed：定义虚线边框（浏览器中为实线）
  4. solid：定义实线
  5. double：定义双线，宽度取决于border-width
border-right/left-color：左右边框颜色
  1. color：定义左右边框的颜色
  2. transparent：指定边框颜色透明

  4. length：自定义右边框宽度

margin（外边距）：(完成)

margin-bottom/top（上下边距）

  1. auto：浏览器设置的上/下外边距
  2. length：定义固定的上/下外边距，默认值为0

padding（内边距）：（完成）

text（文本）：（完成）

text-align-last：对齐文本最后一行

常用方法：

  1. auto：最后一行被调整，并向左对齐（默认值）
  2. left：最后一行向左对齐
  3. right：最后一行向右对齐
  4. center：最后一行居中对齐
  5. start：最后一行在行开头对齐
  6. end：

  1. left：排列左边对齐（默认）

  4. justify：两端对齐文本效果
  5. inherit：继承父元素

text-decoration-line：顶部显示线条

常用方法：

  1. none：文本修饰没有线条（默认值）
  2. underline：文本下方显示条线
  3. overline：文本上方显示一条线
  4. line-through：文本中间显示一条线
  5. inherit：继承父元素
<!DOCTYPE html>

text-decoration-style：下方显示波浪线

常用方法：

  1. solid：默认值，线条将显示为单线
  2. double：线条显示双线
  3. dotted：线条将显示为点状线
  4. dashed：线条显示为虚线
  5. wavy：线条显示为波浪线
  6. inherit：继承父元素

代码示例：

p{
   text-decoration:underline;
   -moz-text-decoration-style:wavy;
}

text-indent：缩进段落第一行：

代码示例：

p{
   text-indent:50px
   text-indent:50%;
}

text-justify：改变字与字之间间距

常用API：

  1. auto：浏览器决定齐行算法
  2. none：禁用齐行
  3. inter-word：增加/减少单词间间隔
  4. inter-ideograph：用文意文本来排齐内容
  5. inter-cluster：只对不包含内部单词间隔内容进行排齐
  6. distribute：类似报纸版面
  7. kashida：通过拉伸字符来排齐内容。
div{
  text-align:justify;
  text-justify:inter-word;
}

text-overflow：文本溢出包含它元素发生

常用API：

  1. clip：修建文本
  2. ellipsis：显示省略符号来代表被修建的文本

代码示例：

p{
  text-overflow:clip;
}

text-shadow：文本阴影效果：

常用API：

  1. h-shadow：必须，水平阴影位置
  2. v-shadow：必须，垂直阴影位置
  3. blur：可选，模糊距离
  4. color：可选阴影颜色

代码示例：

h1{
  text-shadow:5px 5px 5px ##FF0000;
}

text-transform：转换不同元素文本

常用方法：

  1. none：带有小写字母和大写字母标准文本
  2. capitalize：文本中的每个单词以大写字母开头
  3. uppercase：定义仅有大写字母
  4. lowercase：定义无大写字母，仅有小写字母

代码示例：

h1{
  text-transform:uppercase;
}
h2{
  text-transform:capitalize;
}
p{
  text-transform:lowercase;
}

常用方法：

  1. xx-small
  2. x-small

  5. large

  • %：设置基于父元素百分比

font-family：字体系列

  • family-name - 指定的系列名称：具体字体的名称，比如：”times”、”courier”、”arial”。
  • generic-family - 通常字体系列名称：比如：”serif”、”sans-serif”、”cursive”、”fantasy”、”monospace”。

使用某种特定的字体系列（Geneva）完全取决于用户机器上该字体系列是否可用；这个属性没有指示任何字体下载。因此，强烈推荐使用一个通用字体系列名作为后路。

font-style：字体样式

常用API：

  1. normal：标准字体样式
  2. italic：斜体字体样式
  3. bolique：倾斜字体样式

代码示例：

p.normal{
  font-style:normal;
}
p.italic{
  font-style:italic;
}
p.oblique{
  font-style:oblique;
}

<p class -"normal">这是默认文字</p>

font-weight：文字粗细

常用API：

  1. normal：标准字符（默认）
  2. bold：粗体字符
  3. bolder：定义更粗的字符
  4. lighter：定义更细的字符
  5. 数字值：100、200

代码示例：

p.normal{
  font-weight:normal;
}
p.thick{
  font-weight:bold;
}
p.thicker{
  font-weight:900;
}
